# python-virtual-assistant

Python Virtual Assistant with voice command control.

Features
- Speech Recognition
- Voice Command
- Authentication
- Download YouTube Videos
- Get any crypto current price
- Tell current time
- Tell Jokes
- Interact with Chat GPT
